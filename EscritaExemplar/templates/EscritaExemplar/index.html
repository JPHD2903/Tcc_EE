Certamente se você chegou aqui e deve está pensando, que "que tipo de site é esse? Como ele funciona? Oque se pode fazer nele?" Fique atento que respondemos a seguir

Oque é o "EscritaExemplar?"

Para além de um TCC, uma ferramenta utilitária

A proposta do "EscritaExemplar" é ser uma ferramenta rápida e prática para correção de redações, tudo isso sendo feita de maneira automática através de uma API (Interface de Programação de Aplicação) do ChatGPT. Isso possibilita uma maior dinamicidade na hora de corrijir redações, visto que você equanto usuário pode definir prompts específicos para defenir a forma como sua redaçãoserá corrigida, do tema mais simples, até os critérios do ENEM, o "EscritaExemplar" tem tudo oque você precisa para corrijir aquela redação (ou aomenos deveria).

Mas porque fazer um site desses?

Praticidade, rapidez e versatilidade

Certamente o advindo das Inteligências Artificais está provocando grandes mudanças no cotidiano de todos,não a toa a ferramenta do ChatGPT vem se popularizando cada vez mais. Com isso em mentes pensamos em nos aproveitar dessa ferramenta para um ramo específico e bastante presente no mundo do estudo, visto que redações representam umaspecto bastante presente nesse âmbito. Para além disso, o "EscritaExemplar" serve também comoferramenta de auxílio tanto para discentes quanto para docentes.
